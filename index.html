My Projects

Lunar Lander

In the Lunar lander project, I use a Python script to set up a reinforcement learning agent using the TensorFlow Agents library to solve the LunarLander-v2 environment from the OpenAI Gym. Then, I set up the environment using the LunarLander-v2 game from the OpenAI Gym. The observation and action specifications of the environment are printed to the console.Next, I create the Q-Network with two fully connected layers of 512 and 256 neurons, respectively. After that, I create the DQN agent with the Q-Network and other hyperparameters such as the optimizer, loss function, discount factor, and exploration strategy.

I create the replay buffer, which stores experiences of the agent as I interact with the environment. Then, I add the replay buffer observer to my driver, which is responsible for collecting experiences from the environment. Finally, I train the agent for a given number of iterations using the driver, and I log the training progress to the console. The final policy learned by the agent is saved to a file.

Face Recognition Attendence System

A Face recognition attendance system is used in various industries to save from fake presents. In this system, the Harr-cascade algorithm is used. It captures the image of the user and recognizes the face from the saved database, if it is matched it fills the attendance in the excel sheet. If any new user comes for the first time, the system takes his/her photographs and saved them to the directory in 12 x 12 format. when we click on the train image, it starts to detect the image from various perspectives and learn from that. when the next time user comes, they don't need to do anything. just by seeing the face, the system fills the attendance.

This is a project based on python and machine learning. This was a group project led by me. we were three members in the group. in this project, I had taken care of the front-end as well as back-end. Tkinter was used to make an interface. It is very comfortable to use python in machine learning projects because in python we have to code less.
